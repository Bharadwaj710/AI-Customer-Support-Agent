# Dashboard Troubleshooting

## Dashboard Not Loading

If the dashboard does not load:

1. Refresh the page.
2. Clear browser cache.
3. Check internet connection.
4. Try incognito mode.

## Missing Widgets

If widgets are missing:

1. Refresh dashboard layout.
2. Enable widgets from settings.

## Slow Dashboard

If dashboard is slow:

1. Reduce active filters.
2. Close unused tabs.
3. Refresh browser.
